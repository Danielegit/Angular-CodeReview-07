Home

Welcome to Phone Book, an example of web application, developed using Angular, Firebase and Bootstrap which lets you to write down your phone contacts, without the need to waste paper, directly with your laptop or your smartphone.

Important Contacts

Here you can find the section important contacts, a page that has been written directly in html (it is a the static content), where is possible to find the most important contact that you have.

My Phone Book

This is the heart of the application, here you'll find a dinamic application with which is possible to write down new contacts, modify or delete those already existing within the database.
